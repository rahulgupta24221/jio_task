      #   Task Management System

  A task management system to manage the tasks and its status.

  # Overview

  This task management system is built using the Nest.js framework and MongoDB as a database. It allows you to create, update, and delete tasks and manage their statuses.

  # Features

  Create, update and delete tasks.
  Assign task to a person.
  Change task status between to-do, in-progress, and completed.
  Used Basic auth




## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
